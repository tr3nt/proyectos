## Projects

### Installation

1. Run composer
```bash
composer install
```
2. Put DB connection values in **.env** file

3. Run migrations
```bash
php artisan migrate
```
```bash
php artisan db:seed
```

4. Run NPM
```bash
npm install
```
```bash
npm run dev
```
